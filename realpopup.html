houtBox

Tweet using the Chrome address bar

Get Started

0. Make sure you are signed in here.
1. Type "@" in the address bar then tap the spacebar
2. Type in your tweet
3. Hit enter to send the tweet

About the Developers

ShoutBox was created by two Developers from Edmonton, Alberta who sought out a way to make live-tweeting from the desktop a more enjoyable experience.
